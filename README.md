# Program 6a - Factorial Servlet

## Objective
Create a servlet to calculate factorial of a number using HTML form.

## Files
- FactorialServlet.java
- index.html

## How to Run
1. Run project on Apache Tomcat
2. Open: http://localhost:8080/Program6a/
3. Enter number and submit
